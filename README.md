# deloitte-data-analytics-project

Deloitte Data Analytics Project (Forage)
📌 Project Overview

This project was completed as part of the Deloitte Data Analytics Virtual Experience Program on Forage. It focuses on solving real-world business problems using data analysis and visualization.

The project involves working with JSON data and Excel datasets to generate insights and build interactive dashboards using Tableau.

🎯 Objectives
Analyze telemetry data to identify machine failure patterns
Perform gender pay gap analysis to uncover inequality
Transform raw data (JSON & Excel) into meaningful insights
Create interactive Tableau dashboards for visualization

🛠️ Tools 
JSON,Microsoft Excel,Tableau,GitHub

🔍 Description

Analyzed telemetry data (in JSON format) to understand machine behavior and identify potential failure patterns.

📈 Key Insights
Detected abnormal patterns leading to device failures
Identified high-risk machines based on performance data
Observed trends in operational metrics over time
💡 Outcome
Helps in enabling predictive maintenance and reducing system downtime.

👩‍💼 Gender Pay Gap Analysis
🔍 Description

Used Excel dataset to analyze salary distribution and identify pay disparities.

📈 Key Insights
Identified gender-based salary differences
Compared pay across roles and departments
Highlighted inequality trends
💡 Outcome
Supports better decision-making for fair compensation policies.

📊 Tableau Dashboard
🔍 Features
Interactive and user-friendly visuals
